**1. 메인 구조(전체 페이지 트리)**

RiderFlow
├── 로그인(Login)
├── 회원가입(Sign Up) / (선택)
├── 대시보드(Dashboard)
│     ├── 정산 업로드(Upload Settlement)
│     ├── 정산 결과(Result Overview)
│     ├── 기사별 정산 상세(Rider Details)
│     ├── 지난 정산 내역(Historical Reports)
│     ├── 기사 관리(Rider Management)
│     ├── 지사 설정(Branch Settings)
│     └── 알림센터(Notification Center)
└── 로그아웃

**2. 대시보드 메인 페이지 상세 설계**

### **대시보드 목적**

- 업로드된 정산의 요약 정보 제공
- 기사별 수익/건수/추가 비용 등 핵심 데이터 표시
- 빠르게 정산 페이지로 이동할 수 있는 허브 역할

대시보드 UI 와이어프레임(텍스트 형태)

# **3. 대시보드 구성 요소(컴포넌트 명세)**

## 3-1. 정산 요약 카드(Summary Cards)

| 카드 | 포함 정보 | 설명 |
| --- | --- | --- |
| 총 배달 건수 | 이번 주 기사 전체 건수 | 업로드된 Excel 기준 |
| 총 정산액 | 전체 기사 정산 합 | 자동 계산 |
| 기사 수 | 정산 대상 기사 | 엑셀에서 자동 분리 |
| 미발송 정산표 | 아직 발송되지 않은 정산표 | 추후 문자/카톡 기능과 연결 |

---

## 3-2. 상위 수익 라이더 TOP 5

### 기능

- 기사 이름 + 수익 + 총 건수 표시
- 클릭 시 “기사별 정산 상세 페이지”로 이동

### 데이터

- 수익 기준 내림차순
- 기본 5개, 지사 설정에서 조정 가능

---

## 3-3. 최근 업로드한 정산 파일

### 표시 항목

- 파일명
- 업로드한 일자/시간
- 정산 기간 (예: 12/01 ~ 12/07)
- 처리 완료 여부
- 오류 여부

### 기능

- "정산 보기" 버튼
- "다시 업로드" 버튼

---

## 3-4. 기사별 최근 정산 테이블

### 테이블 컬럼 예시

| 기사명 | 배달 건수 | 총 수익 | 기타 비용 | 최종 정산액 | 상세 보기 |
| --- | --- | --- | --- | --- | --- |

### 기능

- 정렬 (수익/건수)
- 검색 (이름/전화번호)
- 상세 페이지 이동

# 📄 **4. 주요 기능 상세 명세서 (Functional Specification)**

---

# 📌 4-1. 엑셀 업로드 기능

### 필수 기능

- 엑셀(XLS, XLSX) 업로드
- 데이터 자동 파싱
- 전화번호/이름/라이더ID 기준 자동 매칭
- 기사별 정산 데이터 분리
- 업로드 오류 감지(누락된 열/포맷 오류 등)

### 고급 기능

- 컬럼 매핑 기능(지사마다 파일 형식 다를 경우)
- 업로드 템플릿 제공

---

# 📌 4-2. 정산 결과 페이지

### 제공 정보

- 전체 요약
- 기사별 요약 목록
- 필터(날짜/기사명/금액 범위)

### 기능

- PDF/엑셀 다운로드
- 정산표 일괄다운로드
- “기사별 상세 보기” 링크

---

# 📌 4-3. 기사별 정산 상세 페이지

### 내용

- 프로필(이름/전화번호)
- 기간별 정산 그래프
- 모든 정산 항목 상세 테이블
- 정산서 PDF 생성 버튼

---

# 📌 4-4. 지난 정산 내역 페이지

### 기능

- 날짜별 파일 리스트
- 검색/필터
- 재다운로드 가능
- 파일 다시 분석 가능

---

# 📌 4-5. 기사 관리 페이지

### 기능

- 기사 자동 등록(엑셀 기반)
- 수동 등록 가능
- 기사 정보 수정(계좌번호/연락처 등)
- 정산 히스토리 연결

---

# 📌 4-6. 지사 설정

### 기능

- 지사 프로필 관리
- 정산서 로고/컬러 커스텀
- 사용자 권한 관리(직원/관리자)
- 정산 기본 공식 설정 (예: 공제율, 위험부담금)
